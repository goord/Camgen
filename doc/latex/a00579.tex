\subsection{Detailed Description}
\subsubsection*{template$<$class T, std\+::size\+\_\+t N$>$class Camgen\+::vector$<$ T, N $>$}

Fixed-\/size vector class. 

Because Camgen uses many vector objects which have their sizes determined at compile-\/time, this class increases the performance as it allows compilers to perform optimisations.

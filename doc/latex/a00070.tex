\subsection{Detailed Description}
\subsubsection*{template$<$class model\-\_\-t, std\-::size\-\_\-t N\-\_\-in, std\-::size\-\_\-t N\-\_\-out$>$class Camgen\-::\-C\-M\-\_\-algorithm$<$ model\-\_\-t, N\-\_\-in, N\-\_\-out $>$}

Amplitude-\/computing algorithm class template. 

Core class of the Camgen library for evaluating scattering amplitudes of arbitrary processes in any Q\-F\-T. The first template parameter denotes the model class, the second and third the respective number of incoming and outgoing particles.

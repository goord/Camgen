14  * Declarations and definitions of the evaluate and cfd_evaluate class *
15  * templates. The evaluate class takes a vertex type (possibly, a composed type) *
16  * as its template argument, and incorporates the corresponding recursive *
17  * equations. These are however absent in the default definition, and therefore *
18  * each usable vertex class should also contain a specialisation of the evaluate *
19  * class template with the static functions first, second, third and fourth, *
20  * denoting the recursions and a static vector-valued member function *
21  * get_index_ranges, filling the argument vector with the correct index ranges *
22  * of the amplitudes involved in the combination. *
23  * *

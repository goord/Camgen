Detailed Description

template<class model_t>
class Camgen::model< model_t >

model base class template.

All user-model classes should be derived from model<derived-model-type>. This class has no data members or non-static member functions. It serves only as an interface between the derived model_t implementation and the hidden model_wrapper<model_t> class templte, which contains the vertex and particle content.

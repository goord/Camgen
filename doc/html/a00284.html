Detailed Description

template<class T, class comp_type = std::less<T>>
class Camgen::histogram< T, comp_type >

Histogram-producing class template.

The first template argument denotes the type of the random number, the second argument a comparison class.

Constructor: the first argument is a pointer whose value will be written to the histogram each time the store-function is called.

The second argument is optional and provides a speedup for large samples, when taken to be the expected sample size.

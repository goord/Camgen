14  * Euclidean spacetime definition. For models in dimensions >0, spacetime *
15  * typedef is required for the inner products of momenta and Lorentz vectors. A *
16  * spacetime type wraps an implementation metafunction, taking the value type *
17  * and dimension as parameters and defining the metric tensor and inner product *
18  * between real- or complex-valued vectors. Note that in the present version of *
19  * Camgen, only scalar wave function are defined in Euclidean signature *
